# 1 定义

Web API是用于Web服务器或Web浏览器的应用程序编程接口（API）。 这是一个Web开发概念，**通常仅限于Web应用程序的客户端（包括正在使用的任何Web框架）**，因此通常不包括Web服务器或浏览器实现细节，如SAPI或Web浏览器引擎API
